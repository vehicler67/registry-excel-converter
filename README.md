# 등기부등본 PDF → Excel 변환기 v1.0

**HanaXellOcr0.7 역공학 기반 — 제2 독립 도구**

---

## 개요

대한민국 법원 등기부등본 PDF를 OCR로 읽어들여  
HanaXellOcr0.7과 동일한 **6시트 구조의 Excel 파일**로 변환합니다.

**제1 도구** (`budongsan_test/parser_v7.py`)와 달리 이 도구는:
- **완전 독립 실행** — 폴더째 복사해서 어느 맥에서나 즉시 사용 가능
- **Excel 네이티브 출력** — 병합셀·스타일·컬럼너비 완전 재현
- **한국어 각주 완비** — 모든 함수에 역공학·실패사례·주의사항 기록

## 사용법

### 방법 1: 더블클릭 (추천)
```
실행.command  더블클릭 → PDF 선택 → 자동 변환 → Excel 열기
```

### 방법 2: 터미널
```bash
python3 변환기.py                # 파일 선택 다이얼로그
python3 변환기.py 등기부.pdf      # 직접 경로 지정
python3 변환기.py 등기부.pdf -o 결과.xlsx  # 출력 경로 지정
```

## 출력물

| 시트 | 내용 |
|------|------|
| 전체요약 | 기본정보 + 현황 요약 (85행×8열) |
| 갑구 | 소유권에 관한 사항 |
| 을구 | 소유권 외 권리 (근저당, 전세권 등) |
| 공동담보목록 | 여러 필지에 걸친 담보 목록 |
| 요약-갑구 | 갑구 요약 테이블 |
| 요약-을구 | 을구 요약 테이블 |
